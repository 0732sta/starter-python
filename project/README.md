### Bar Tab Calculator
- locate **project** folder
```bash
python
```
- import class Tab in [bar_tab.py](https://github.com/0732sta/starter-python/blob/master/project/bar_tab.py) 
```bash
from bar_tab import Tab
```
- create new instance, for example table1
```bash
table1=Tab()
```
```bash
table1
```

